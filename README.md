# Space Raiders

Space Raiders is a small Unity space survival game where you fly through asteroid fields, avoid enemy ships, and try to stay alive as long as possible.

---

The project focuses on a few simple systems that work well together: custom ship movement with boost, roll, and banking, procedural asteroid generation around the player, and enemy ships that chase and pressure you while you move through the map.

---

It started as a local hackathon project with friends, and this version is an attempt to turn that original idea into something more complete and playable. Detached from original [Space Raiders repo](https://github.com/Sgtryan10/space-raiders.git).
